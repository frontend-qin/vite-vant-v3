# vite-vant-v3

#### 仓库
```bash
https://github.com/frontend-qin/vite-vant-v3.git
```

### install

```bash
yarn
```

### 脚本
```bash

# 本地服务接口
yarn dev

# 启动测试服务接口
yarn test

# 启动正式服务接口
yarn prod

# 打包：本地开发环境
yarn devbuild

# 打包：测试服务器环境
yarn testbuild

# 打包：正式服务器环境
yarn build

# 打包后预览
yarn serve

```
### 配置接口环境

[关于vite环境变量](https://cn.vitejs.dev/guide/env-and-mode.html);

```bash
# 配置本地开发接口地址
.env.development

# 配置正式环境接口地址
.env.production

# 配置测试环境接口地址
.env.test

```
